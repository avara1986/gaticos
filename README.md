# Gaticos

Gaticos ("Little Cats"? "Cutey Cats"?) it's a simple package to print "miau" in your console or print an ascii
draw of a cat in your shell with quotes ¯\_(ツ)_/¯

```shell
pip install gaticos
```

Print miaus:

```shell
gaticos -c 2
>> Miau!
>> Miau!
```

Print motivation frase in spanish:

```shell
gaticos animame
```
